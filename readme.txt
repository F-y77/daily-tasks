[h1]饥荒联机版每日任务系统[/h1]

这个模组为饥荒联机版添加了每日任务系统，让游戏更加有趣和有挑战性！

[h2]特点[/h2]
[list]
[*] 每天随机分配任务
[*] 多种任务类型：采集、狩猎、砍树、采矿、钓鱼、烹饪等
[*] 完成任务后获得丰厚奖励
[*] 可自定义任务数量、难度和奖励倍数
[*] 按 T 键查看当前任务
[*] 按 Y 键查看任务进度
[/list]

[h2]任务类型[/h2]
[list]
[*] [b]采集任务[/b] - 采集浆果、胡萝卜、芦苇、蘑菇、花朵、蜂蜜、树枝等
[*] [b]狩猎任务[/b] - 猎杀兔子、蜘蛛、猪人、蜜蜂、青蛙、浣熊、高鸟、触手等
[*] [b]砍树任务[/b] - 砍倒普通树木、松树、桦树、大树等
[*] [b]采矿任务[/b] - 开采石头、金矿、大理石、硝石、燧石、冰等
[*] [b]钓鱼任务[/b] - 钓普通鱼、大鱼、海钓、特殊鱼等
[*] [b]烹饪任务[/b] - 烹饪普通食物、高级食物、肉类食物、素食料理、海鲜料理等
[*] [b]生存任务[/b] - 存活一整天、保持健康/理智/饱腹状态、寻找宝藏、探索新区域、制作工具、建造房子等
[/list]

[h2]配置选项[/h2]
[list]
[*] [b]每日任务数量[/b] - 设置每天分配的任务数量
[*] [b]任务难度[/b] - 调整任务的难度
[*] [b]奖励倍数[/b] - 调整任务奖励的数量
[*] [b]显示通知[/b] - 开启或关闭任务通知
[/list]

[h2]使用方法[/h2]
[list]
[*] 安装模组后，每个玩家在游戏中会自动获得每日任务
[*] 每天早上会分配新任务
[*] 按 T 键查看当前任务详情
[*] 按 Y 键查看任务完成进度
[*] 完成任务后会自动获得奖励
[/list]

[h2]兼容性[/h2]
[list]
[*] 仅支持饥荒联机版
[*] 服务器和所有客户端都需要安装此模组
[/list]
